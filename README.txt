Library Management System
Overview
The Library Management System is a console-based application built using C++. It enables efficient management of library inventory and operations, including book borrowing, returning, and tracking overdue books.

Features
Book Inventory Management:

Add new books to the library database.
Search for books by title, author, or unique ID.
Check the availability of specific books.
Student Borrowing System:

Borrow and return books while maintaining accurate records.
Track overdue books and calculate fines.
Student-specific borrowing histories for reference.
Automation:

Automatic due-date calculation based on borrowing date.
Availability checks prevent double-booking of the same resource.
Technology Stack
Language: C++
Data Structures: Linked lists for dynamic storage of books and borrowing records.
Modular Design: Functions separated into modules for easy maintenance and scalability.
How to Run
Clone or download the project repository.
Open the project folder in your preferred IDE or text editor.
Compile and run the main.cpp file.
Follow the on-screen prompts to interact with the system.
Possible Extensions
Adding a graphical interface.
Integrating a database for persistent storage.
Online user portal for accessing book details and borrowing status remotely.
